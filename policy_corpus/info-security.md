# Information Security Policy

**Version:** 1.0  
**Date:** March 10, 2026  

## 1. Framework
Aligned with ISO 27001 principles and NDPA security requirements.

## 2. Key Controls
- Asset management  
- Access control (MFA mandatory)  
- Encryption in transit/rest  
- Vulnerability scanning (weekly)  
- Nigerian network considerations (power redundancy, DDoS protection)  

## 3. Incident Response
Team activation within 1 hour; NDPC notification per GAID.
