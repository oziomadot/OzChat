# Employee Code of Conduct **Version:** 1.0 **Effective Date:** March 10, 2026 ## Purpose This Code outlines expected behavior for all NSR employees, contractors, and interns. ## 1. Compliance with Laws Employees must comply with Nigerian laws, including the Nigeria Data Protection Act 2023 and GAID 2025. ## 2. Ethical Standards - Act with integrity - Maintain confidentiality of user data - Avoid conflicts of interest ## 3. Workplace Behavior - Respect diversity (ethnicity, gender, age, religion) - Zero tolerance for harassment - Promote inclusive language ## 4. Data & Asset Protection Never share user personal data (location, age, gender, preferences) outside authorized systems. Report breaches immediately. ## 5. Reporting Violations Use anonymous whistleblower channel or report to HR/Compliance Officer. **Acknowledgment:** All employees must sign annually.
